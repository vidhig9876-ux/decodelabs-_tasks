# 🛡️ Cybersecurity Internship Projects

## 📌 About

This repository contains the projects I completed during my **Cybersecurity Internship at DecodeLabs**. The objective of these projects was to understand fundamental cybersecurity concepts through hands-on Python programming.

Each project focuses on a different area of cybersecurity, including password security, encryption, phishing awareness, and system security auditing.

---

## 🚀 Projects Included

### 🔐 Project 1 – Password Strength Checker

A Python application that evaluates password strength based on:

- Password Length
- Uppercase Letters
- Numbers
- Special Characters

**Features**

- Password Score
- Weak / Medium / Strong Classification
- Improvement Suggestions

📁 Folder:
`Project-1-Password-Strength-Checker`

---

### 🔒 Project 2 – Caesar Cipher Encryption

A beginner-friendly implementation of Caesar Cipher.

**Features**

- Encrypt Messages
- Decrypt Messages
- User-defined Shift Key

📁 Folder:
`Project-2-Caesar-Cipher`

---

### 🎣 Project 3 – Phishing Awareness Analyzer

Analyzes email or message content for common phishing indicators.

**Features**

- Detects Phishing Keywords
- Displays Red Flags
- Safe / Suspicious / Highly Suspicious Classification

📁 Folder:
`Project-3-Phishing-Awareness-Analyzer`

---

### 🛡️ Project 4 – System Security Audit Tool

Performs a basic security assessment of a computer system.

**Checks**

- Firewall
- Antivirus
- Operating System Updates
- Strong Password
- Two-Factor Authentication

**Features**

- Security Score
- Risk Level
- Security Recommendations

📁 Folder:
`Project-4-System-Security-Audit`

---
